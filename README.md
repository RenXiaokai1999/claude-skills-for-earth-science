## ✅ 快速安装

将本仓库的 `.claude/skills` 复制到你的项目根目录：

```
<your-project>/.claude/skills/
```

## 🧩 技能清单（功能卡片）

**地学相关**
- geopandas — GIS 矢量数据处理  
- pymatgen — 矿物/晶体结构分析  
- astropy — 坐标/时间/行星计算  

**通用科研工具**
- openalex-database — 跨学科文献检索  
- literature-review — 综述框架与结构化整理  
- scientific-writing — 论文写作与改写  
- citation-management — 引用与 BibTeX 管理  
- markitdown — 文档转 Markdown  
- scientific-slides — 学术汇报幻灯片草稿  
- document-skills — PDF/DOCX/PPTX/XLSX 处理  
- matplotlib — 科研级静态图  
- seaborn — 统计可视化  
- plotly — 交互式图表  
- statsmodels — 统计建模  
- scikit-learn — 机器学习  
- sympy — 符号计算  
- networkx — 网络/图分析  
- dask — 大数据并行处理

## 🧪 使用示例（学术风格）

**示例 1（地学）**  
使用 `geopandas` 读取 `D:\data\faults.shp`，投影到 EPSG:32649，计算断层长度并输出 CSV。

**示例 2（文献）**  
使用 `openalex-database` 搜索 “loess stratigraphy China”，返回近 5 年高被引论文清单。
